优化博客性能

发表于 2016-05-07

今天发现博客首页的加载速度比较感人，于是就用 Google 出品的 Pagespeed Insights 检测了一下，发现了不少问题，而且这些问题都严重拖慢了加载时间。

  • HTML / CSS / JS / Photos 都是未压缩状态

  • CSS 出现阻塞，即首页出现空白的情况
