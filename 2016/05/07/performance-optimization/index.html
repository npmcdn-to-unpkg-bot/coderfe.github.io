优化博客性能

发表于 2016-05-07   |  

今天发现博客首页的加载时间比较慢，于是就用 Google 出品的 Pagespeed Insights 检测了一下，发现了不少问题，而且这些问题都严重拖慢了加载时间。以下：

  • HTML / CSS / JS / Photos 都是未压缩状态
  • 部分静态资源没有设置缓存
  • CSS 出现阻塞，即首页出现空白的情况
